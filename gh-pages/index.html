ESP8266 Auto Update (OTA)

Librería para la actualización automática (OTA) de placas ESP8266 apuntando a un repositorio. Administra tu firmware de forma remota y sencilla.

📖 1. Cómo usar esta librería

Para integrar esta librería en tu proyecto de Arduino/PlatformIO:

  1. Descarga o clona este repositorio en la carpeta de librerías de tu entorno.
  2. Incluye la cabecera principal en tu código: #include "ESP8266AutoUpdate.h"
  3. Inicializa la clase con la URL de tu repositorio donde se aloja el archivo .bin y el archivo de versión.
  4. Llama al método de comprobación dentro de tu setup() o periódicamente en el loop().
#include <ESP8266AutoUpdate.h>

void setup() {
    Serial.begin(115200);
    // Tu lógica de conexión WiFi aquí...
    
    // Iniciar AutoUpdate (ejemplo hipotético)
    AutoUpdate.begin("https://raw.githubusercontent.com/TU_USUARIO/TU_REPO/main/version.json");
    AutoUpdate.check();
}

void loop() {
    // Tu código principal
}

🔀 2. Cómo Clonar y Forkear el Repositorio

Hacer un Fork (Bifurcar)

Si deseas realizar modificaciones propias o contribuir a este proyecto:

  1. Ve a la esquina superior derecha de la página de GitHub.
  2. Haz clic en el botón Fork.
  3. Esto creará una copia del proyecto en tu propia cuenta de GitHub.

Clonar el Repositorio

Para descargar el código a tu computadora local, usa Git en tu terminal:

git clone https://github.com/AlejandroBalonga/esp8266-auto-update.git

Si hiciste un Fork, reemplaza AlejandroBalonga por tu nombre de usuario.

🔌 3. Herramienta Web de Configuración y Flasheo

Conecta tu ESP8266 por USB. Utiliza navegadores basados en Chromium (Chrome, Edge, Brave).

Paso A: Flashear Firmware Base

Utiliza ESP Web Tools para cargar el firmware base inicial en la placa.

Paso B: Configurar ESP (WiFi y OTA URL)
